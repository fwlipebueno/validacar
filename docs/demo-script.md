# Roteiro da Demo
Roteiro objetivo para uma apresentação de 3 minutos. A fala deve ser direta e mostrar produto funcionando, não apenas explicar a ideia.

## 1. Problema
O CAR reúne dados importantes do imóvel rural, mas nem sempre deixa claro o que o produtor ou técnico deve fazer antes de uma retificação. A informação existe, mas aparece espalhada entre status, análise, demonstrativo, mensagens e restrições.

## 2. Solução
O ValidaCAR é um assistente de pré-retificação. Ele organiza dados do imóvel, mostra o mapa, identifica pontos de atenção, explica os riscos em linguagem simples e gera um checklist antes de tentar corrigir ou reenviar o cadastro.

O CAR já mostra os dados. O ValidaCAR mostra o que fazer com eles.

## 3. Demonstração
1. O usuário seleciona um imóvel de exemplo.
2. O sistema mostra resumo e status do cadastro.
3. O mapa destaca inconsistências e sobreposições simuladas.
4. Os alertas explicam os riscos em linguagem simples.
5. O checklist mostra o que corrigir, separar ou aguardar.
6. O relatório final organiza o diagnóstico antes da retificação.

## 4. Diferencial
O ValidaCAR não é só um validador de mapa. Ele conecta dados cadastrais, análise visual e orientação prática. A experiência ajuda o usuário a entender prioridade, risco e próxima ação antes de entrar em um fluxo sensível de retificação.

## 5. Fechamento
Com o ValidaCAR, o usuário não precisa apenas ver dados técnicos do CAR. Ele entende o que está acontecendo e qual é o próximo passo.
